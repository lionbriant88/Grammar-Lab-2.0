# Grammar Lab - 语法实验室

英语语法实验室是一款离线英语教学辅助软件，专注于帮助学生和教师理解英语语法结构。

## 功能

- **时间轴分析** - 可视化动词时态的时空关系
- **句剖析分析** - 句子结构和语法成分的详细分析
- **句扩展分析** - 从简单句到复杂句的渐进式扩展示例

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 打包
npm run build:win  # Windows
npm run build:mac  # macOS
npm run build:linux  # Linux
```
